EvoSuite 测试生成器

上传 .class、.jar 或 .java 文件生成测试用例

{% with messages = get_flashed_messages(with_categories=true) %} {% if messages %} {% for category, message in messages %}
{{ message }}
{% endfor %} {% endif %} {% endwith %}

支持的格式:

  • .class 文件 - 将自动提取类名并生成测试
  • .jar 文件 - 将分析整个 JAR 文件
  • .java 文件 - 将先编译再生成测试
